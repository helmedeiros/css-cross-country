# css-cross-country

Code written while working through Code School's **CSS Cross-Country** course.

A learning record covering CSS fundamentals — external stylesheets, selectors
(ID, compound, descendant), the box model, positioning and typography — kept for
the historical trail.

## Layout

Each `level_N/` folder holds the exercises for that level, one numbered
sub-folder per exercise, each with its own `index.html` and `style.css`.
